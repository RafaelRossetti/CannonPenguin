# Project Yeti Launch (Cannon Penguin)

Um jogo de arcade viciante onde você ajuda o Yeti a lançar o pinguim o mais longe possível!

## Como Jogar

1. **Lançamento (QTE):** No menu inicial, clique em "LANÇAR". Uma barra de força começará a oscilar. Pressione **ESPAÇO** ou **CLIQUE** quando o indicador estiver na zona amarela (80%) para o impulso máximo!
2. **Voo:** Use as setas **Esquerda/Direita** ou as teclas **A/D** para inclinar o pinguim.
   - **Inclinar para baixo:** Aumenta a velocidade, mas perde altitude rapidamente.
   - **Inclinar para cima:** Dá sustentação (lift) temporária, mas reduz a velocidade.
3. **Impulsos:** Procure por nuvens de vento (boosters aéreos) e trampolins de gelo para renovar sua energia e ir mais longe.
